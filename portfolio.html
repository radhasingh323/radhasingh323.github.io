In this project we decided to create a solar system to demonstrate our knowledge on movement. We have 4 planets being drawn one by one and it ends with another loop in which a moon orbits the very last planet.

In this project we essentially allow user input through the use of the onkeypress function in which the user is able to click "S" and shoot the ball. We incorporated movement by the constant horizantal movement of the hoop and also added a leaderbaord and countdown function.
